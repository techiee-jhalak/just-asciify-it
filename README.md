# 🧩 JUST-ASCIIFY-IT

An interactive, browser-based terminal workspace that converts webcams, images, and video feeds into matrix-style ASCII character streams in real time.

📡 **Live Deployment Link:** [https://techiee-jhalak.github.io/just-asciify-it/](https://techiee-jhalak.github.io/just-asciify-it/)

---

## ⚡ Features

*   **Real-Time Processing:** Instantly translates live webcam frames or video uploads into smooth ASCII streams.
*   **Granular Render Controls:** Custom sliders to dynamically modify typography size, contrast matrix levels, and overall illumination gain.
*   **Industrial Export Tools:** 
    *   **`.TXT` Capture:** Downloads raw structural layout matrix data directly.
    *   **High-Res `.PNG` Canvas:** Generates high-resolution raster print configurations of your render.

---

## 🎨 Render Configurations

### 1. Character Matrix Patterns
You can switch the structural density framework using the pattern selector:
*   **Simple (`simple`):** Minimalist text weight mapping (` .:-=+*#%@`).
*   **Complex (`complex`):** Extended high-fidelity shading arrays for deep tonal range.
*   **Binary (`binary`):** Pure telemetry representation using exclusively `0` and `1` streams.
*   **Blocks (`blocks`):** Legacy terminal retro emulation mapping density to solid block fragments (` ░▒▓█`).

### 2. Telemetry Color Profiles
Change the global render matrix on the fly with built-in color channels:
*   🟢 **Matrix Green:** Classic command line phosphor emulation.
*   🔵 **Cyber Cyan:** Bright neon terminal overlay.
*   💖 **Neon Pink:** High-contrast synthwave aesthetic.
*   🟣 **Deep Purple:** Deep dark web background shading.
*   🔴 **Crimson Red:** Emergency override notification hue.
*   🟡 **Neon Yellow:** High-visibility hazard readout.
*   🌈 **RGB Mode (Cyber):** True multicolor mapping that samples RGB values directly from source frames.
*   ⚪ **Black & White:** Clean, raw monochrome text arrays.
