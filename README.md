# 👁️ JUST-ASCIIFY-IT

An interactive, browser-based terminal workspace that converts webcams, images, and video feeds into matrix-style ASCII character streams in real time. 

---

## ⚡ Features

*   **Real-Time Processing:** Instantly translates live webcam frames or video uploads into smooth ASCII streams.
*   **Granular Render Controls:** Custom sliders to dynamically modify typography size, contrast matrix levels, and overall illumination gain.
*   **Typography Sets:** Toggle instantly between variable-density text matrices (`Simple`, `Complex`, `Binary`, or digital `Blocks`).
*   **HUD Visualizer Themes:** Swap interfaces profiles across multiple telemetry profiles (`Matrix Green`, `Cyber Cyan`, `Neon Pink`, `RGB Mode`, and more).
*   **Industrial Export Tools:** 
    *   **`.TXT` Capture:** Downloads raw structural layout matrix data directly.
    *   **High-Res `.PNG` Canvas:** Generates high-resolution raster print configurations of your render.
